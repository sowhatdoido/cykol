Cykol Carousel

Created as a technical skills test for TeamOne. This first carousel should meet all the requirements asked for the submission.

DIV Carousel

I wanted to make a carousel that could handle plain img tags or divs with text and such. The above slider has images nested inside divs because I wanted to apply a loader icon and specific styles, but using just plain img tags would have worked as well.

Custom Effects

You can achieve custom affects by editing `duration` and `easing`, as well as modifying CSS. Here I change the easing to linear, turn up the duration to half a second, and removed the scaling on the thumbnails but decreased the opacity.

Callback

I added support for callbacks on initiation and slide change (That's actually how I get the text to change in the first example without moving it). In this example, I'm firing an alert when you reach the end of the verse.
